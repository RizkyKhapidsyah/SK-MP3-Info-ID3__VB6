Title: clsMP3
Description: A class module that can read ID3v1, ID3v2, and header information from MP3s. Also includes a function for searching dirs for MP3's and subdirs. Can be compiled into an ActiveX Dll for use in ASP or other types of apps. Sample project shows proper use of the class.
